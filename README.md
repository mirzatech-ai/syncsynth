## How the modules work
Open any `.html` directly in a browser (or load into nw_wrld). Click **Microphone** (analyses live audio)
or **Load Track** (drop an mp3/wav — it both plays and drives the visuals). Bass/mid/high bands drive
distinct parameters. Fullscreen canvas, translucent HUD with intensity/hue controls, "SyncSynth" footer.
